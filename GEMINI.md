# GEMINI.md — vybezverse blog writer

Πριν γράψεις οποιοδήποτε νέο blog post, διάβασε **[BLOG_AGENTS.md](BLOG_AGENTS.md)** ολόκληρο. Δεσμευτικό, όχι προαιρετικό.

Σύνοψη κανόνων που παραβιάστηκαν στο παρελθόν (μην τα ξανακάνεις):
- **Release ≠ Beat.** Ένα single δεν είναι "beat-σειρά". Μπερδεύτηκαν στο post για το "Jazz Bar των τεράτων".
- Ημερομηνία στο `date` = πραγματική ημερομηνία γεγονότος, όχι σημερινή by default. Το Jazz Bar κυκλοφόρησε Halloween, όχι σήμερα.
- Μην εφευρίσκεις naming/σειρές (πχ "Vintage Freq" ως σειρά beats) — αυτό ήταν λάθος, δες BLOG_AGENTS.md.
- Κάθε post για release πρέπει να έχει link `[Κυκλοφορίες](/releases)` για αγορά, plus τα πραγματικά streaming links από το `Releases.jsx`/`releases.json` — ποτέ εικασία link.
- Αν δεν ξέρεις fact (ημερομηνία, link, τίτλος) → `[THEO CHECK: ...]`, ποτέ εικασία.
